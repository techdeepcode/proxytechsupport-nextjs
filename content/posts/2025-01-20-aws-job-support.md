## AWS Job Support – Real-Time Cloud Engineering Help

Deploying to AWS is never as simple as it looks on paper. Permissions errors, VPC misconfigurations, Lambda cold starts, ECS task failures — these problems cost hours. Our **AWS job support** service connects you directly with a certified AWS engineer who will work through the problem with you live.

### AWS Services We Support

**Compute & Containers**
- EC2 – AMI selection, auto-scaling groups, launch templates, spot instances
- Lambda – cold starts, layers, concurrency limits, VPC configuration
- ECS / Fargate – task definitions, service discovery, ALB integration
- EKS – node groups, Helm charts, IAM roles for service accounts

**Storage & Databases**
- S3 – bucket policies, presigned URLs, lifecycle rules, cross-region replication
- RDS / Aurora – parameter groups, read replicas, failover, Performance Insights
- DynamoDB – partition key design, GSI, on-demand vs. provisioned, streams
- ElastiCache – Redis cluster setup, eviction policies, connection pooling

**Networking & Security**
- VPC – subnets, route tables, NAT gateway, VPC peering, Transit Gateway
- IAM – least-privilege policies, roles for EC2/Lambda/ECS, SCPs
- CloudFront – origins, behaviours, cache invalidation, signed URLs
- API Gateway – REST vs. HTTP API, throttling, authorizers, CORS

**IaC & CI/CD**
- CloudFormation – stacks, nested stacks, drift detection
- Terraform – modules, state management, workspaces, remote backends
- CodePipeline / CodeBuild – build stages, artifact management
- GitHub Actions with AWS OIDC – keyless deployments
