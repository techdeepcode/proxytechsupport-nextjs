## What We Help With

### UI Automation Frameworks

**Selenium (Java, Python, C#)**
- Framework architecture – Page Object Model, Fluent interface, factory patterns
- WebDriver setup – ChromeDriver, GeckoDriver, cross-browser matrix
- Explicit vs. implicit waits – eliminating flaky tests
- TestNG / JUnit test organisation, parallel execution
- Selenium Grid for distributed testing

**Playwright**
- JavaScript / TypeScript / Python implementations
- Fixtures and test isolation
- API request interception and mocking
- Visual comparison testing – snapshot testing, pixel comparison
- Trace viewer – debugging test failures step by step
- GitHub Actions / Azure DevOps integration

**Cypress**
- Custom commands and plugins
- Stubbing and intercepting XHR requests
- Component testing alongside E2E
- Cypress Cloud for parallel test execution
- Dealing with dynamic selectors and shadow DOM

### API Testing

- Postman / Newman – test scripts, environment chaining, CI runs
- REST Assured (Java) – schema validation, authentication chains, response assertions
- Karate – BDD-style API testing with minimal code
- Contract testing with Pact – consumer-driven contracts between microservices
- Performance testing APIs with k6 and Gatling

### Mobile & Accessibility Testing

- Appium setup for iOS and Android – capability configuration, real device vs. emulator
- WCAG 2.1 compliance testing – axe-core integration, screen reader testing
